# Urban-Air-Quality-Classification-in-Bekasi-using-SVM-XGBoost-and-Random-Forest
Conducted a machine learning analysis on air quality in Bekasi, focusing on major pollutants (PM2.5, PM10,  NO₂, CO, O₃, SO₂). Implemented data preprocessing, model development, and SMOTE technique to handle imbalanced datasets, achieving improved classification accuracy.
